# Dependency Injection in Spring Framework

This project demonstrates the different types of Dependency Injection in Spring Framework with simple and practical examples.

## 📌 Concepts Covered

### ✅ Constructor Injection

Dependencies are injected through the constructor.
Recommended for mandatory dependencies and improves immutability.

### ✅ Setter Injection

Dependencies are injected using setter methods.
Useful for optional dependencies.

### ✅ Field Injection

Dependencies are injected directly into fields using annotations.
Simple to use for smaller applications and learning purposes.

---

## 🚀 Why Dependency Injection?

Dependency Injection helps in:

* Reducing tight coupling between classes
* Improving code maintainability
* Making unit testing easier
* Building scalable and flexible applications
